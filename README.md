### Comment lancer le projet

1.PHP

- **Tout d'abord, télécharger et installer php version 8.4 https://www.php.net/downloads.php?os=windows&osvariant=windows-downloads&version=8.4**


2. Cloner le projet

- **git clone https://github.com/Reyan403/CheckInSystem
cd GestionLicence**


3. Créer la base de donnée

[check_in_system.sql](https://github.com/user-attachments/files/29160128/check_in_system.sql) 

Cliquez pour télécharger la base de donnée, puis dans l'interface MySQL ou autre, importez ce fichier.


4. Démarrer l'application

- **Marquer dans le lien de la recherche de google localhost/php/CheckInSystem/public/ afin de voir l'application dans son entièreté.**

### Suggestions d'améliorations avec plus de temps 

Si le temps imparti l'avait permis, j'aurais souhaité implémenter les fonctionnalités suivantes :

- **Filtre de recherche avancé : Mise en place d'un système de recherche dynamique dans la vue hiérarchique afin de retrouver instantanément un employé par son nom ou son prénom.**
- **Module de gestion du personnel : Développement d'une interface d'administration dédiée permettant d'ajouter, de modifier ou de supprimer facilement des profils d'employés.**
- **Régularisation des pointages : Création d'une fonctionnalité permettant aux managers de corriger l'historique des actions (par exemple, rectifier un "départ" cliqué par inadvertance à la place d'une "arrivée").**
- **Suivi des temps de pause : Intégration de nouveaux statuts pour gérer de manière précise les départs et retours de pause des salariés.**
- **Calcul des heures de travail : Implémentation d'un module automatisant le calcul du temps de travail effectif de chaque employé sur une période donnée.**
- **Sécurité renforcée : Sécurisation de l'authentification par l'ajout d'un mécanisme anti-brute force, verrouillant temporairement la saisie du code PIN après plusieurs tentatives erronées.**
